# วัตถุประสงค์
 * เว็บไซต์นี้เน้นออกแบบให้ทุกคนและทุกวัยใช้ได้โดยมีการออกแบบที่เน้นเข้าใจง่ายไม่ยุ่งยากและเป็นภาษาไทย
 * มีสูตรอาหารที่หลากหลายและต้องบอกแหล่งที่มาของวัตถุดิบเสมอ

<br/>

# รายละเอียดเพิ่มเติม
  * หากต้องการแสดงความคิดเห็นหรือส่งรีเควส จำเป็นต้องทำการเข้าสู่ระบบก่อนเสมอ
  * เว็บไซตร์นี้สร้างโดยการใช้ Next.js และ tailwind เป็นหลัก
  * Database ที่เลือกใช้คือ mongodb atlas
